BASIC FRACTALS
# Tested for mac user (El capitan realease)

### Installation
```sh
$ git clone https://github.com/overedge/fractol
$ cd fractol
$ make
$ make clean
```
### Usage
```sh
$ ./fractol <name_of_fractal> julia, mandelbrot, burningship
```
### Commands
| Usage       | Commands |
| ------------- | ------------- |
| LOCK MOUSE  | X|
| RESET  | R |
| COLOR  | C  |
| ZOOM KEYBOARD |  O/L  |
| ITTERATION INCREASE  |  I/K |
| CHANGE FRACTALS  | ENTER  |
| MOVE  | ARROW ← ↑ →  |
